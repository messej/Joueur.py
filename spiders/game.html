Game¶

class games.spiders.game.Game¶

Bases: joueur.base_game.BaseGame

The class representing the Game in the Spiders game.

There’s an infestation of enemy spiders challenging your queen broodmother spider! Protect her and attack the other broodmother in this turn based, node based, game.

current_player¶

The player whose turn it is currently. That player can send commands. Other players cannot.

Return type:games.spiders.player.Player
current_turn¶

The current turn number, starting at 0 for the first player’s turn.

Return type:int
cut_speed¶

The speed at which Cutters work to do cut Webs.

Return type:int
eggs_scalar¶

Constant used to calculate how many eggs BroodMothers get on their owner’s turns.

Return type:float
game_objects¶

A mapping of every game object’s ID to the actual game object. Primarily used by the server and client to easily refer to the game objects via ID.

Return type:dict[str, games.spiders.game_object.GameObject]
get_game_object(id)¶

gets the game object with the given id, or None

Returns:BaseGameObject in the game with the given id, or None if not found
initial_web_strength¶

The starting strength for Webs.

Return type:int
max_turns¶

The maximum number of turns before the game will automatically end.

Return type:int
movement_speed¶

The speed at which Spiderlings move on Webs.

Return type:int
nests¶

Every Nest in the game.

Return type:list[games.spiders.nest.Nest]
players¶

List of all the players in the game.

Return type:list[games.spiders.player.Player]
session¶

A unique identifier for the game instance that is being played.

Return type:str
spit_speed¶

The speed at which Spitters work to spit new Webs.

Return type:int
weave_power¶

weave_speed¶

Return type:int
webs¶
