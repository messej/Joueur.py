Checker¶

class games.checkers.checker.Checker¶

Bases: games.checkers.game_object.GameObject

The class representing the Checker in the Checkers game.

A checker on the game board.

game_object_name¶

String representing the top level Class that this game object is an instance of. Used for reflection to create new instances on clients, but exposed for convenience should AIs want this data.

Return type:str
id¶

A unique id for each instance of a GameObject or a sub class. Used for client and server communication. Should never change value after being set.

Return type:str
is_mine()¶

Returns if the checker is owned by your player or not.

Returns:True if it is yours, False if it is not yours.
Return type:bool
kinged¶

If the checker has been kinged and can move backwards.

Return type:bool
log(message)¶

Adds a message to this GameObject’s logs. Intended for your own debugging purposes, as strings stored here are saved in the gamelog.

Parameters:message (str) – A string to add to this GameObject’s log. Intended for debugging.
logs¶

Any strings logged will be stored here. Intended for debugging.

Return type:list[str]
move(x, y)¶

Moves the checker from its current location to the given (x, y).

Parameters:
  • x (int) – The x coordinate to move to.
  • y (int) – The y coordinate to move to.
Returns:

Returns the same checker that moved if the move was successful. None otherwise.
